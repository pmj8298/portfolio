워드클라우드 프로젝트

프로젝트 개요

한글 텍스트 데이터에서 명사를 추출해 워드클라우드를 생성하고, 주요 단어들의 빈도수를 시각화

핵심 코드

from wordcloud import WordCloud
from konlpy.tag import Komoran
import matplotlib.pyplot as plt

text = '''(텍스트 내용)'''

komoran = Komoran()
nouns = komoran.nouns(text)
nouns_text = ' '.join(nouns)

wordcloud = WordCloud(
    font_path='D2Coding-Ver1.3.2-20180524.ttf', 
    width=800, height=400, background_color='lightgrey'
).generate(nouns_text)

plt.figure(figsize=(10,5))
plt.imshow(wordcloud, interpolation='bilinear')
plt.axis('off')
plt.show()

분석 결과

명사 단어들을 추출해 단어 빈도수를 세어, 이를 기반으로 워드클라우드를 생성하여 데이터 내 주요 키워드를 시각화

시각화 이미지

느낀 점 및 배운 점

텍스트 전처리와 자연어 처리 기법을 실제 데이터에 적용해보고, 시각화를 통해 데이터를 직관적으로 표현하는 방법을 배움
