공간정보 시각화

1. 개요

브이월드에서 제공하는 행정구역 공간 데이터를 활용하여 GeoPandas와 Matplotlib를 이용해 부산 지역의 행정구역 단위 공간 시각화를 수행했습니다.

shapefile(.shp) 데이터를 로드하고 각 구의 이름에 따라 다른 색상으로 시각화하여 데이터의 분포와 특성을 파악할 수 있도록 구성하였습니다.

2. 사용 기술

  • Python (3.10)
  • GeoPandas, Matplotlib
  • 행정구역 공간 데이터 (.shp)

3. 주요 코드

import geopandas as gpd
import matplotlib.pyplot as plt

shp_path = './ref/LARD_ADM_SECT_SGG_26_202502.shp'
gdf = gpd.read_file(shp_path, encoding='EUC-KR')

fig, ax = plt.subplots(1,1,figsize=(13,10))
gdf.plot(column='SGG_NM', ax=ax, legend=True, cmap='tab20')
plt.show()

5. 결론

공간정보 시각화는 지역별 분석을 효과적으로 수행할 수 있도록 도와주며, 특히 데이터 기반 의사결정이나 시각적 인사이트 제공에 유용합니다. 향후 다른 지역으로 확장하거나 공공 API와 연동한 실시간 지리 데이터 분석으로도 확장 가능성이 높습니다.
